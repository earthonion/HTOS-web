How It Works

1

Upload your save

Grab the save from your console via FTP and upload it here. We accept both PS4 and PS5 save formats — encrypted or decrypted.

2

Pick an operation

Choose what you need — decrypt, encrypt, resign, re-region, or create a new save. Set any options like the target account ID or region.

3

A worker handles it

Your job gets sent to a real PS4 or PS5 console running in the background. The console processes the save using the actual system libraries — no emulation, no guesswork.

4

Download the result

Once it's done (usually a few seconds), download your processed save and FTP it back to your console. That's it.

Frequently Asked Questions

What do I need to use this?
A jailbroken PS4 or PS5 with FTP access so you can pull saves off the console. You'll also need a free account here. That's really it — no special software on your PC required.
Is this free?
Yep, totally free. The processing is done by community-contributed consoles. If you want to help keep things running, you can contribute your own console as a worker or sponsor the project.
Which firmware versions are supported?
For PS4, any jailbreakable firmware works (up to 13.00). For PS5, currently firmware versions that support the PS5 jailbreak (up to 10.01). The worker consoles handle all the crypto, so your save doesn't need to match the worker's firmware.
How does the worker system actually work?
People in the community volunteer their jailbroken consoles to run a small background program (the "worker"). When you submit a job, one of these consoles picks it up, processes the save using Sony's own system libraries, and sends the result back. It's like a distributed network of PlayStations doing the heavy lifting.
Can I use saves from someone else's account?
That's exactly what the resign feature is for. Upload the save, enter your own account ID, and the save gets re-signed so your console thinks it belongs to you. Works across PS4 and PS5.
What if no workers are online?
Your job sits in the queue until a worker comes online. Usually there are consoles available, but during off-hours it might take a bit longer. You can check the worker status on the dashboard after logging in.
How can I contribute a worker?
If you've got a jailbroken PS4 or PS5 that's online most of the time, you can run the garlic worker on it. Create an account, grab a worker key from the Contribute page, and follow the setup instructions. Your console will process jobs in the background — you can still use it normally.

Open Source

Garlic Saves is fully open source. The worker code, the server, the save tools — all of it is on GitHub. Built by the homebrew community, for the homebrew community.
