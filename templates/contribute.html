Worker Setup (PS4 & PS5)

Download garlic-worker from GitHub — grab garlic-worker-ps4.elf or garlic-worker-ps5.elf for your console.

  1. Make sure your console is jailbroken and running a payload loader — PS4 elfldr / PS5 elfldr (port 9021)
  2. Generate a worker key above
  3. Via FTP, create the directory /data/garlic/
  4. Save your config as /data/garlic/config.ini (or launch the worker once and it will create a default config to edit)
  5. Send the ELF to your console: cat garlic-worker-ps4.elf | nc -w 2 <CONSOLE_IP> 9021
  6. The worker will show a notification and start polling for jobs
  7. Recommended: For PS5, add the ELF to your Y2JB autoloader.txt. For PS4, add it to your Vue-After-Free config.json. This way the worker runs automatically in the background while you play games
  8. Tip: Increase pollInterval in your config (e.g. 60 or 120) to reduce resource usage
  9. Tip: Go to Settings > Power Save Settings and set your console to not enter rest mode automatically, so the worker stays running
